# The Isle Evrima Interactive Map

A high-resolution, interactive web map for *The Isle Evrima*, built with React and Leaflet. 

**[Click here to view the live map!](https://balhun.github.io/evrima_map_viewer/)**

## Features
* **High-Resolution Imagery:** Supports up to 8K map textures.
* **In-Game Coordinate Tracking:** Paste your in-game coordinates (X, Y) directly from the game, and the math automatically translates them into web coordinates to drop a pin exactly where your dinosaur is standing.
* **Map Clicker:** Click anywhere on the map to get the in-game X and Y coordinates for that exact spot.
* **Layer Toggles:** To turn Mud and Water overlays on or off.
